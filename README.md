## ✨ Key Features

- **Premium 3D Floating Navbar**: Sophisticated glassmorphism design with dual shadows, gold glow accents, and interactive link lifts for immersive navigation.
- **Responsive Menu Grid**: Pixel-perfect cards with 3D hover animations (elevate & scale), showcasing exquisite dishes with prices and seamless 'Order Now' CTAs.
- **Butter-Smooth Scrolling**: Effortless section transitions anchored by the sticky nav.
- **Interactive Reservation Modal**: Luxe contact form with validation, plus integrated booking overlay for instant table reservations.
- **Testimonials Carousel**: Auto-rotating slider with star ratings for authentic guest voices.
- **Fully Responsive**: Optimized for all devices, from mobile to ultra-wide desktops.

---

## 🎨 Design Philosophy

Embracing a **'Pixel Perfect'** ethos, The RessWitchers transcends traditional web design through strategic depth and dimensionality. The dark theme (#1a1a1a base) punctuated by opulent gold (#D4AF37) and crisp white accents evokes timeless luxury.

- **3D Immersion**: Multi-layered shadows, strategic `translateY` hovers, and gold glows create tactile depth, transforming flat pixels into a premium experience.
- **Glassmorphism Mastery**: Backdrop blurs and translucent overlays for modern sophistication.
- **Minimalist Elegance**: Clean grids, ample whitespace, and serif/sans typography harmony ensure focus on content without clutter.

Every interaction—be it a menu card lift or nav glow—delivers refined feedback, mirroring the precision of haute cuisine.

## 🚀 Installation & Usage

1. **Clone/Download** the repository.
2. **Open `index.html`** directly in any modern browser.
3. **Live Development** (recommended):
   ```
   npx live-server .
   ```
   Auto-reloads on changes.

No dependencies. Instant preview.

---

## ☁️ Deployment

Deploy effortlessly to static hosts:

- **Netlify**: Drag `index.html` to [netlify.com/drop](https://app.netlify.com/drop).
- **GitHub Pages**: Push to `gh-pages` branch.
- **Vercel**: `vercel --prod`.

Custom domain ready. HTTPS by default.
